## Software Engineer | GenAI & Agentic Systems | Full-Stack Developer | Cloud-Native Builder

I'm a Software Engineer with **4+ years of experience** building production-grade web applications in Python and Java, now focused on integrating **Generative AI** — LLMs, RAG, multi-agent systems, evals, and MCP — into real, customer-facing products. I hold a Master's in Computer Science from **Stevens Institute of Technology**, and I care deeply about code quality, comprehensive testing, observability, and owning the full lifecycle from design through deployment and on-call ownership.
